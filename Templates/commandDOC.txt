.TH MVTIT 1 "$(date + "%d %B %Y)" "1.0" "Move Title - A tool for renaming terminal windows"

.SH NOM
mvtit - MVTIT

.SH SYNOPSIS
mvtit [OPTIONS] [ARGUMENTS]

.SH DESCRIPTION
mvtit is a command-line tool for renaming terminal windows. It allows users to easily change the title of a terminal window to a specified name.

.SH OPTIONS
\fB-h, --help\fR
Display help message and exit.

.SH EXAMPLES
Rename the current terminal window to "My Window":
\fBmvtit "My Window"\fR
